<!-- Installation Guide -->
<div align="center">
  <h2>
    <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/docker/docker-original.svg" width="30" height="30" alt="Installation">
    🔧 安装指南
  </h2>
</div>

<details>
<summary><b>📋 环境要求</b></summary>

- **Node.js** >= 16.x
- **npm** 或 **yarn**
- **Git**

</details>

<details>
<summary><b>🚀 快速开始</b></summary>

### 1. 克隆项目
```bash
git clone <repository-url>
cd blog
```

### 2. 安装后端依赖
```bash
cd server
npm install
# 或
yarn install
```

### 3. 配置环境变量
在 `server` 目录下创建 `.env` 文件：
```env
DATABASE_URL="file:./prisma/dev.db"
JWT_SECRET="your-jwt-secret-key"
CORS_ORIGIN="http://localhost:5173"
```

### 4. 初始化数据库
```bash
# 生成Prisma客户端
npx prisma generate

# 运行数据库迁移
npx prisma db push

# (可选) 填充种子数据
npx prisma db seed
```

### 5. 安装前端依赖
```bash
cd ../web
npm install
# 或
yarn install
```

</details>

<div align="center">
  <h2>
    <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/swagger/swagger-original.svg" width="30" height="30" alt="API">
    📚 API文档
  </h2>
</div>

<details>
<summary><b>🔐 认证相关</b></summary>

| 方法 | 端点 | 描述 |
|------|------|------|
| `POST` | `/api/auth/register` | 👤 用户注册 |
| `POST` | `/api/auth/login` | 🔑 用户登录 |
| `GET` | `/api/auth/verify` | ✅ 验证Token |

</details>

<details>
<summary><b>📝 文章管理</b></summary>

| 方法 | 端点 | 描述 |
|------|------|------|
| `GET` | `/api/posts` | 📋 获取文章列表 |
| `GET` | `/api/posts/:slug` | 📖 获取文章详情 |
| `POST` | `/api/posts` | ✍️ 创建文章 (管理员权限) |
| `PUT` | `/api/posts/:id` | ✏️ 更新文章 (管理员权限) |
| `DELETE` | `/api/posts/:id` | 🗑️ 删除文章 (管理员权限) |

</details>

<details>
<summary><b>🏷️ 分类和标签</b></summary>

| 方法 | 端点 | 描述 |
|------|------|------|
| `GET` | `/api/categories` | 📂 获取分类列表 |
| `GET` | `/api/tags` | 🏷️ 获取标签列表 |
| `POST` | `/api/categories` | ➕ 创建分类 (管理员权限) |
| `POST` | `/api/tags` | ➕ 创建标签 (管理员权限) |

</details>
